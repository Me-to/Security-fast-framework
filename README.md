Security的前后端页面不分离快速开发脚手架  
开箱即用,只需配置yml里的配置即可快速搭建属于自己的权限认证框架  



### 技术选型：

|框架|名称|
|:---|:---|
|核心框架|Spring boot 2.3.3|
|安全框架|Spring Security  2.3.3|
|缓存数据库|Redis |
|数据库|MySql 8.0|
|持久层框架|Mybatis  2.1.3|
|数据库连接池|Druid  1.1.22|
|页面交互|Thymeleaf（我没有使用，建议使用）|
|实体类插件|lombok|
|验证码生成工具|Spring Socia  1.1.6|
|工具类|Hutool|
